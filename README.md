# cmake-exercise

## Overview
```
$ tree
.

├── README.md
```

## How to use
```
$ git clone git@github.com:rkmathi/cmake-exercise
$ cd cmake-exercise

### Make
$ cmake . && make

### Execute
$ ./bin/main.out
```
